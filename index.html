Proyectos

Sistema de Inventario para Pequeñas Empresas

Implementé un sistema de inventario utilizando Python y Django. El sistema permite a las pequeñas empresas gestionar sus productos, realizar seguimientos de stock y generar reportes de ventas. Utilicé SQLite como base de datos y creé una interfaz de usuario intuitiva con Bootstrap.

Aplicación Móvil para Seguimiento de Hábitos

Desarrollé una aplicación móvil multiplataforma con Flutter que permite a los usuarios realizar un seguimiento de sus hábitos diarios. La aplicación ofrece recordatorios, estadísticas y un sistema de recompensas. Los datos se sincronizan en tiempo real con Firebase, lo que permite a los usuarios acceder a su información desde cualquier dispositivo.

Portal Educativo con Evaluaciones en Línea

Creé un portal educativo utilizando Laravel para permitir a los profesores crear y administrar cursos, y a los estudiantes acceder a evaluaciones en línea. Implementé un sistema de calificación automática y un panel de administración para gestionar usuarios y contenidos. La interfaz fue desarrollada con Vue.js y Tailwind CSS
